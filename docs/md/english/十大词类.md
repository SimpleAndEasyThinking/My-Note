# 动词

动词具有五大句型

* 主语+动词
* 主语+动词+宾语
* 主语+动词+间接宾语+直接宾语
* 主语+动词+宾语+(宾语)补语
* 主语+动词+(主语)补语(表语)

> 句子成分为十大词性，动词、冠词、介词、名词、代词、数词、形容词、副词、连词、叹词。

简单句子形式，就是“主语”+“谓语”

*主语* :一般是人或物，不管抽象还是具体。

*谓语*:广义上——动作/发生了什么事。“动作”指广义上的“动词”。

## 简单句

1. 可以独立完成的动作

主语+不及物动词

不及物动词：没有被动者的动词

2. 有一个被动者

主语+单及物动词+宾语

像“喜欢”这样的动词，没有被动者就没有实际意义。。

3. 有两个被动者

主语+双及物动词+间接宾语+直接宾语

既有直接宾语又有间接宾语的动词。
双及物动词 ：有两个动作承受者。
对某对象执行具体事件，被执行者为间接宾语（ indirect object ），具体事件为直接宾语（ direct object ）

4. 只有一个被动者（需补充）

主语+复杂及物动词+宾语+（宾语）补语

复杂及物动词 ： 需要有补语的动词

5. 非“动作”

主语+系动词+（主语）补语/表语

系动词： 赋予主语某种性质状态的“划等号”的动词

主语补语在中国英语教学中称之为“表语”（ predicative ）

总结 : 英语中所有句子，当拆到不能再拆后，都是在说，“什么 + 怎么样”。
“什么”对应“主语”，“怎么样”对应“谓语”。
谓语都有一个核心动词，叫作“谓语动词”。以上便是谓语动词的五种类型

主谓宾结构：主语+谓语动词+宾语

谓语 ≠ 谓语动词 ，句子主语后面的余下句子部分就是谓语。谓语动词只是谓语的一部分。

句子其他元素：

> 定语：用来修饰主语或宾语

> 状语：用来修饰谓语动词

> 同位语： 用来再把主语或宾语说一遍

## 复合句

compound sentences（句子简单叠加，是并列关系，直接翻译为“复合句”，有的语法书是“并列句”）

## 复杂句

complex sentences（把一个句子套在另一个句子里，甚至一个句子套的一个句子还能再套一个句子，无限下去，是从属关系，直接翻译为“复杂句”） 。
