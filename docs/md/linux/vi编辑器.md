
# vi文本编辑器

文本编辑器的作用

* linux系统中的配置文件大多数以文本文件的形式保存于各个目录
* linux系统管理员通过修改配置文件进行系统管理

文本编辑器的分类

* 行编辑器与全屏幕器编辑器
* 字符界面编辑器与图形界面编辑器
