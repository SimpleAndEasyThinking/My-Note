# 工具

## ftp工具

- ftp文件传输协议，应用层协议，用于网络文件传输
- 用于从ftp服务器_上传或者下载文件
- ftp_工具支持两种传输模式: ASCII模式（文本传输）和Binary模式（二进制传输）
操作流程

1. 建立到ftp服务器的连接
2. 用户身份验证
3. 查看文件列表 dir
4. [删除文件] del
5. 设置文件传输模式 binary
6. 下载文件 get
7. 上传文件 put
8. 关闭退出 bye
